# Self-Diagnostic Protocol v1.0
## 文書番号：MOCKA-PHASE9-SDP-v1.0
## 区分：自己診断プロトコル（追記のみ可）

---

# 1. 目的
MoCKA 文明体系が自らの状態を定期的に診断し、  
文明構造・文脈継承・文明間連携・外部脳統合の健全性を  
自律的に評価するための制度的プロトコルを定義する。

---

# 2. 前提条件
- Self-Referential Protocol v1.0 が稼働している  
- Autonomous Update Log により自律更新が開始されている  
- 文脈継承（Phase5）が安定している  
- 三文明（Alpha・Beta・Gamma）が統合されている  

---

# 3. 自己診断の対象
自己診断は以下の文明構造を対象とする：

### 3.1 文明構造の健全性
- 三文明の役割分担  
- 文明間連携の整合性  
- 文明体系図との一致  

### 3.2 文脈継承（Phase5）
- 文脈の連続性  
- 文脈の切断の有無  
- 文脈追加の整合性  

### 3.3 外部脳統合（Gamma）
- 外部脳との同期状態  
- 情報流通の健全性  
- 外部脳負荷の適正性  

### 3.4 自律更新（Phase9）
- 更新頻度の適正性  
- 更新対象の妥当性  
- 更新ログの整合性  

---

# 4. 自己診断プロセス
Self-Diagnostic Protocol v1.0 における診断は以下の手順で行う：

1. 文明体系の全体スキャン  
2. 文脈継承の整合性チェック  
3. 文明間連携の評価  
4. 外部脳統合の状態確認  
5. 自律更新履歴の分析  
6. 診断結果の記録（Self-Diagnostic Log）  

---

# 5. 診断頻度
- 初期フェーズ：必要に応じて随時  
- 安定フェーズ：文明更新ごと  
- 永続化フェーズ（Phase10）：自動化を前提とする  

---

- 文明体系の永続化プロトコル（Phase10）の準備
