# MoCKA-Alpha — Overview
## 文書番号：MOCKA-DERIVED-CIV-README-v1.0
## 区分：派生文明文書（追記のみ可）

MoCKA-Alpha は Genesis Protocol によって生成された最初の派生文明であり、  
MoCKA 文明体系における「決定事項の長期保管」と「外部協働」を担う基幹文明である。

---

# 1. 文明の概要
MoCKA-Alpha は、MoCKA における意思決定・知識・経験を長期的に保存し、  
外部組織・外部AI・外部人材との協働を可能にする文明である。

---

# 2. 文明の目的
- 決定事項の長期保管  
- 外部協働のための透明な知識基盤の提供  
- 文脈を失わない意思決定の継承  

---

# 3. 文明生成の背景
英知成長の記録と、得た知識・経験を外部へ流用・活用するための文明として設計された。  
MoCKA の知識体系を文明として保存し、外部協働を可能にするための基盤を整備する。

---

# 4. 文脈継承（Phase5）
- MoCKA の制度文書体系  
- 決定事項の記録文化  
- 文脈継承（Phase5）の厳格な運用  
- 外部協働を前提とした透明性  
- 英知成長を文明として蓄積する思想
