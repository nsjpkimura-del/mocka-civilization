# Phase6：再考文明（Reconsideration Civilization）
## 文書番号：MOCKA-PHASE6-v1.0
## 区分：文明制度正本（改変不可）

本書は、文明が自己修正・再考・再評価を行うための
制度的枠組みを定義する文明制度文書である。

---

# 1. 目的
文明が過去の判断・議論・評価を再検討し、  
必要に応じて改善・修正を行うための手続きを提供する。

---

# 2. 再考の基本原則

- **非破壊性**：過去の記録を改変してはならない  
- **透明性**：再考の理由・根拠・過程を明示する  
- **事実性**：再考は事実に基づく  
- **中立性**：評価・優劣・成功／失敗の断定は禁止  
- **継承性**：再考は文脈継承（Phase5）と整合する  

---

# 3. 再考の対象

- 判断（Phase2）  
- 熟議記録（Phase3）  
- 評価記録（Phase4）  
- 文脈継承の適用範囲（Phase5）  
- 文明の制度運用全般  

---

# 4. 再考プロセス

1. 再考対象の明確化  
2. 関連記録の収集  
3. 再考理由の整理  
4. 再考案の提示  
5. 再考案の熟議（Phase3 に準拠）  
6. 再考結果の確定  
7. 再考記録の保存（事実のみ）  

---

# 5. 運用規定

- 本文書は改変不可  
- 再考記録は事実のみ  
- 再考は文明の改善のために行う  
- 再考は過去の記録を否定するものではない  
- 文脈の切断は禁止  

---

# 6. 付記
本フェイズは文明の「自己修正能力」を保証する制度であり、  
文明の継続的な発展に不可欠である。
