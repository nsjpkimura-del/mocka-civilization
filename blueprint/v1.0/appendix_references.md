# 参考情報（References Appendix）
## 文書番号：MOCKA-APPENDIX-REFERENCES-v1.0
## 区分：文明制度補助文書（追記のみ可）

本書は、Blueprint v1.0 の制度文書を理解・運用するために
参照される補助的情報をまとめた付録文書である。

---

# 1. 文書体系の構造

MoCKA 文明の制度文書は以下の階層で構成される。

- **正本（Canonical Documents）**  
  - Blueprint v1.0  
  - Phase1〜Phase7 文書  
  - principles_and_prohibitions.md  

- **補助文書（Supplemental Documents）**  
  - checklists（3 文書）  
  - appendix（本書を含む）  

- **運用文書（Operational Documents）**  
  - genesis/  
  - lineage-registry/  
  - derived-civilizations/  

---

# 2. 文書間の参照関係

- Phase1〜Phase7 は Blueprint v1.0 を基礎とする  
- checklists は Phase1〜Phase7 の運用を補助する  
- appendix は制度文書の理解を補助する  
- genesis/ は Blueprint v1.0 を参照して文明を生成する  
- lineage-registry/ は生成された文明の履歴を記録する  

---

# 3. 文書の命名規則

- チェックリスト：`checklist_XXXX.md`  
- 付録文書：`appendix_XXXX.md`

---

# 4. 文書の保存規則

- 正本文書は改変不可  
- 補助文書は追記のみ許容  
- 文書の削除は禁止  
- 文書はすべて履歴管理下で保存する

# 6. 運用規定
