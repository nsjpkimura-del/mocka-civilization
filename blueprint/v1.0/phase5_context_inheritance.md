# Phase5：文脈継承（Context Inheritance）
## 文書番号：MOCKA-PHASE5-v1.0
## 区分：文明制度正本（改変不可）

本書は、文明が過去の文脈をどのように継承し、
判断・熟議・評価に反映させるかを定義する制度文書である。

---

# 1. 目的
文明が「過去の文脈」を正しく理解し、  
それを未来の判断・議論・制度運用に適切に反映させるための  
継承手続きを定義する。

---

# 2. 文脈継承の基本原則

- **非恣意性**：文脈は恣意的に切断してはならない  
- **整合性**：過去の記録と矛盾しない形で継承する  
- **透明性**：継承の根拠は明示される  
- **事実性**：文脈は事実に基づく  
- **非評価性**：文脈に対する評価・優劣の記述は禁止  

---

# 3. 文脈継承の対象

- 文明の初期条件（Phase1）  
- 判断基準（Phase2）  
- 熟議記録（Phase3）  
- 評価記録（Phase4）  
- 文明の価値観・制約・環境  
- 文明の制度的履歴  

---

# 4. 文脈継承プロセス

1. 継承対象の明確化  
2. 関連記録の収集  
3. 文脈の整理（事実のみ）  
4. 文脈の適用範囲の決定  
5. 文脈の継承記録の保存  

---

# 5. 運用規定

- 本文書は改変不可  
- 文脈継承は文明の判断の正当性を支える  
- 文脈の切断は禁止  
- 文脈の追加は事実に基づく場合のみ許容  
- 評価・優劣・成功／失敗の記述は禁止  

---

# 6. 付記
本フェイズは文明の「歴史の連続性」を保証する制度であり、  
文明の判断・議論・再考に深く影響する。
