---

# 1. 事前準備チェック

- Blueprint v1.0 が完全に揃っている  
- 文書がすべて改変不可として保存されている  
- lineage-registry/ が存在している  
- genesis/ フォルダが存在している

# 2. 文明生成の入力チェック

- 文明の価値観が定義されている  

---

# 3. 生成プロセスチェック

- Genesis Protocol の手順に従っている

- 評価（Phase4）が中立性を保っている  
- 再考（Phase6）が必要に応じて実施されている

# 4. 出力チェック

- 文明生成ログが保存されている  
- 文明の初期状態が lineage-registry に登録されている  
- 文明生成の根拠が明示されている  
- 文明生成の過程が追跡可能である
