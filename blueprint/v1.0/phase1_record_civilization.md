# Phase1：記録文明（Record Civilization）
## 文書番号：MOCKA-PHASE1-v1.0
## 区分：文明制度正本（改変不可）

本書は、文明の初期状態・目的・制約・環境を記録するための
制度文書である。

---

# 1. 目的
文明の基礎情報を正確に記録し、後続フェイズの判断基盤とする。

---

# 2. 記録項目

- 文明名  
- 文明の目的  
- 文明の初期条件  
- 文明の制約  
- 文明の環境  
- 文明の前提となる価値観  
- 文明の禁止事項（初期版）  

---

# 3. 運用規定

- 本文書は改変不可  
- 追記のみ許容  
- 文明の初期状態は事実のみを記録する  
- 評価・優劣・成功／失敗の記述は禁止  

---

# 4. 付記
本フェイズは文明の「土台」を定義するものであり、  
後続フェイズの判断基準となる。
