# Phase2：決定文明（Decision Civilization）
## 文書番号：MOCKA-PHASE2-v1.0
## 区分：文明制度正本（改変不可）

本書は、文明の意思決定構造・判断基準・優先順位を定義する
制度文書である。

---

# 1. 目的
文明が一貫した判断を行うための基準と手続きを定める。

---

# 2. 決定項目

- 意思決定の階層構造  
- 判断基準（価値・優先順位）  
- 判断の根拠となる情報源  
- 判断の透明性要件  
- 判断の記録方法  

---

# 3. 意思決定プロセス

1. 判断対象の明確化  
2. 必要情報の収集  
3. 判断基準との照合  
4. 合意形成（必要な場合）  
5. 判断の確定  
6. 判断の記録  

---

# 4. 運用規定

- 本文書は改変不可  
- 判断基準は文明全体に適用される  
- 判断の記録は事実のみ  
- 評価・優劣・成功／失敗の記述は禁止  

---

# 5. 付記
本フェイズは文明の「判断の軸」を定義するものであり、  
後続フェイズの熟議・評価に影響を与える。
