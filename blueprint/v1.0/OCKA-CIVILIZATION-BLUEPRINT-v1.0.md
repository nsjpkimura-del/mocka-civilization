# MoCKA Civilization Blueprint v1.0
## 文書番号：MOCKA-BLUEPRINT-v1.0
## 区分：文明制度正本（改変不可）

本書は、MoCKA 文明の制度構造（Phase1〜Phase7）を統合した
文明制度の総合正本である。

本書は以下の目的を持つ。

- 文明の制度的基盤を明確化する  
- 文明生成・運用・継承のための統一基準を提供する  
- 文明の再現性・透明性・継承性を保証する  

本書は改変不可であり、追記のみ許容される。

---

# 1. 文明制度の構造

MoCKA 文明は以下の 7 フェイズから構成される。

1. **Phase1：記録文明（Record Civilization）**  
   文明の初期状態・目的・制約・環境を記録する。

2. **Phase2：決定文明（Decision Civilization）**  
   文明の意思決定構造・判断基準・優先順位を定義する。

3. **Phase3：熟議文明（Deliberation Civilization）**  
   文明内部の議論プロセス・合意形成手続きを定義する。

4. **Phase4：評価文明（Evaluation Civilization）**  
   文明の行動・判断・成果を評価する制度を定義する。

5. **Phase5：文脈継承（Context Inheritance）**  
   文明が過去の文脈をどのように継承するかを定義する。

6. **Phase6：再考文明（Reconsideration Civilization）**  
   文明が自己修正・再考・再評価を行う制度を定義する。

7. **Phase7：協働文明（Collaborative Civilization）**  
   文明が他文明・他主体と協働する制度を定義する。

---

# 2. 文明制度の原則

文明制度は以下の原則に従う。

- **透明性**：すべての制度は公開され、追跡可能である  
- **再現性**：文明は同じ手順で再生成可能である  
- **継承性**：文明は文脈を継承しつつ発展する  
- **非破壊性**：制度文書は改変不可、追記のみ許容  
- **中立性**：評価・優劣・成功／失敗の記述は禁止  
- **事実性**：記録は事実のみを扱う  

---

# 3. 文明制度の禁止事項

文明制度において以下は禁止される。

- 文書の改変（削除・書き換え）  
- 文明の優劣評価  
- 文明の成功／失敗の断定  
- 文脈の恣意的な切断  
- 文明間の序列化
