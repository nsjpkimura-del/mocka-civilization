# Phase3：熟議文明（Deliberation Civilization）
## 文書番号：MOCKA-PHASE3-v1.0
## 区分：文明制度正本（改変不可）

本書は、文明内部の議論プロセス・合意形成手続きを定義する
制度文書である。

---

# 1. 目的
文明が多様な視点を統合し、合理的かつ透明な熟議を行うための
制度的枠組みを提供する。

---

# 2. 熟議の基本原則

- **対等性**：すべての意見は対等に扱われる  
- **透明性**：議論の過程は記録され、追跡可能である  
- **根拠性**：主張には根拠を伴う  
- **非攻撃性**：個人攻撃は禁止  
- **事実性**：事実と推測を区別する  

---

# 3. 熟議プロセス

1. 議題の明確化  
2. 関連情報の提示  
3. 複数視点の収集  
4. 論点の整理  
5. 合意形成の試行  
6. 結論の確定  
7. 議論記録の保存  

---

# 4. 合意形成の方法

- 全会一致  
- 多数合意  
- 条件付き合意  
- 合意不成立時の再考手続き（Phase6 に委譲）  

---

# 5. 運用規定

- 本文書は改変不可  
- 熟議記録は事実のみ  
- 評価・優劣・成功／失敗の記述は禁止  
- 熟議は文明の判断の正当性を支える基盤である  

---

# 6. 付記
本フェイズは文明の「議論の質」を保証するものであり、  
判断・評価・再考のすべてに影響を与える。
