# Phase4：評価文明（Evaluation Civilization）
## 文書番号：MOCKA-PHASE4-v1.0
## 区分：文明制度正本（改変不可）

本書は、文明の行動・判断・成果を評価する制度を定義する
文明制度文書である。

---

# 1. 目的
文明の行動・判断・制度運用を、  
透明かつ中立的な基準に基づいて評価するための枠組みを提供する。

---

# 2. 評価の基本原則

- **中立性**：評価は事実に基づき、価値判断を含まない  
- **透明性**：評価基準・評価過程・評価結果は公開される  
- **再現性**：同じ条件であれば同じ評価結果となる  
- **非序列性**：文明間の優劣をつけることは禁止  
- **非断定性**：成功／失敗の断定は禁止  

---

# 3. 評価項目

- 判断の妥当性  
- 熟議プロセスの適切性  
- 文脈継承の正確性  
- 記録の整合性  
- 制度遵守状況  
- 文明内部の透明性  

---

# 4. 評価プロセス

1. 評価対象の明確化  
2. 関連記録の収集  
3. 評価基準との照合  
4. 評価結果の整理  
5. 評価記録の保存（事実のみ）  

---

# 5. 運用規定

- 本文書は改変不可  
- 評価記録は事実のみ  
- 評価は文明の改善のために行われる  
- 評価は文明の優劣を決めるために使ってはならない  

---

# 6. 付記
本フェイズは文明の「自己理解」を支える制度であり、  
後続フェイズの再考（Phase6）に直接影響する。
