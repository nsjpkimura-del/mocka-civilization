# Replication Protocol v1.0
## 文書番号：MOCKA-REPLICATION-PROTOCOL-v1.0
## 区分：文明複製プロトコル（追記のみ可）

---

# 1. 目的
MoCKA 文明体系（Alpha・Beta・Gamma の三文明連携）を  
別環境・別時代・別リポジトリに複製可能とするための  
正式な手順を定義する。

---

# 2. 前提条件
- 三文明（Alpha・Beta・Gamma）がすべて生成済み  
- 運用開始決定事項が記録済み  
- 外部協働記録が作成済み  
- 文明体系統合記録（Tri-Civilization Integration）が完了している  

---

# 3. 複製対象
以下の文明構造を完全に複製する：

- MoCKA-Alpha（記録文明）
- MoCKA-Beta（生成文明）
- MoCKA-Gamma（統合文明）
- 文明体系図（civilization-map）
- 文明体系統合記録
- 文脈継承（Phase5）に基づく全文書

---

# 4. 複製手順（Replication Steps）
1. 新規リポジトリまたは新規環境を準備する  
2. `civilization-map.md` を最初に配置する  
3. `derived-civilizations/` を作成し、三文明フォルダを複製する  
4. 文書は「追記のみ可」の原則を維持する  
5. 文脈継承（Phase5）を切断しないように複製する  
6. 複製後、`replication-log.md` を作成し、複製の事実を記録する  

---

# 5. 文脈継承の扱い
- 文脈は「複製元 → 複製先」にそのまま継承される  
- 文脈の改変は禁止  
- 文脈の追加は複製後に行う  

---

# 6. 複製後の義務
- `replication-log.md` を作成する  
- 複製先文明の運用開始決定事項を記録する  
- 外部協働記録を作成する  

---
