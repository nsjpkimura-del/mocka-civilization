Core Ideas

This ecosystem treats AI development as an institutional activity, not only software engineering.

The aim is to preserve decision context and verification artifacts so that systems remain auditable over time.

Long-term operation requires memory, rules, and reproducible traces.


How To Read

Start from SYSTEM_OVERVIEW.md.

Then read RESEARCH_POSITIONING.md to understand the intended audience and research framing.

Then read AI_GOVERNANCE_REFERENCE_ARCHITECTURE.md for the structural model.
